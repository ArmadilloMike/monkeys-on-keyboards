# 100 Monkeys on Keyboards

This is a simple webpage that generates random characters/words via monkeys

## Buttons and what they do
| Button               | Description                                           |
|----------------------|-------------------------------------------------------|
| Start                | Starts the generation of chars/words                  |
| Stop                 | Stops the generation of chars/words                   |
| Mode                 | Changes the mode from spam -> words and words -> spam |
| Clear Text           | Clears all output                                     |
| Print Current Output | Opens the browsers print window for the output text   |
| Save Current Output  | Saves the current output as "monkey-output.txt"       |
| (Un)Mute All Sounds  | Mutes or Unmutes all sounds playing                   |
